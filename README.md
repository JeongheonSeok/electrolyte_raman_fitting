# electrolyte_raman_fitting
본 레포지토리의 코드는 spectrum을 lmfit 패키지를 이용하여 deconvolution 하는 코드입니다.

구체적으로는, FSI- 염의 raman spectrum을 free anion, CIP, AGG (AGG1, AGG2,...) 피크로 분해하고, 각 피크의 면적 비교를 하기 위한 코드입니다.

## 의존성
본 코드는 python3 기반입니다. 필요한 패키지는 다음과 같습니다.
```
lmfit
matplotlib

(이하 패키지들은 위 패키지 설치 시 자동 설치됨)
scipy
numpy
```

## 사용법
폴더 및 파일 구조는 유지한 채로, 아래 과정을 따라 실행하면 `results` 폴더에 결과 파일들이 출력됩니다.
### 1. `spectra` 폴더에 스펙트럼 파일들을 넣어둡니다.
피크 피팅을 진행할 스펙트럼 파일들을 폴더에 넣어둡니다.

파일 내용은
```
위치1, 값1
위치2, 값2
...
```
형식이어야 합니다. 예를 들어 다음과 같은 형태입니다.
```
300.00000,1276.47119
302.00000,1193.90417
304.00000,1045.10205
306.00000,883.41663
...
```
### 2. 계산 내용 설정: `peak_location.csv`, `settings.csv`
아래 두 파일의 내용을 필요한 대로 적절히 수정합니다.
#### 가. `peak_location.csv`: 분해할 피크 이름과 위치(중심)
해당 파일에서 입력받은 피크 종류와 중심 위치를 토대로 deconvolution을 진행합니다.

`min_x`와 `max_x` 사이의 범위에서 피크 중심을 탐색하게 됩니다.

아래와 같은 형태로 작성하면 됩니다.
```
name,min_x,max_x
free,719,721
cip,730,734
agg1,740,750
```
**이때, 피크 이름은 띄어쓰기 및 특수문자 없이, 영어 소문자 알파벳과 숫자로만 구성되어야 합니다.**

더 많은 수의 피크로 분해하고 싶다면, 줄을 더 추가하면 됩니다. (단, 너무 많은 피크수는 부정확하거나 무의미한 결과를 얻을 수 있습니다.)

탐색 범위를 완전히 고정할 수는 없습니다. 필요한 경우 최대한 좁은 범위를 입력해 주세요. (e.g. 718.999 ~ 719.001)

#### 나. `settings.csv`: 피크 분해 과정 설정
크게 네 가지 파라미터로 구성되어있습니다.
* `RANGE_MIN`, `RANGE_MAX`: 분해를 진행할 x값 범위
* `PEAK_TYPE`: 피크의 모양을 설정합니다. (0: Gaussian, 1: Lorentzian, 2: Voigt)
* `BACKGROUND`: 배경 보정 여부를 설정합니다. (0: 하지 않음, 1: 상수, 2: 일차함수 1개, 3: 일차함수 2개)

### 3. `raman_fitting.py` 실행
해당 파이썬 파일을 실행하면 `results` 폴더에 피팅 결과 파일들이 생성됩니다.

스펙트럼 파일의 크기에 따라 파일당 수 초~ 수 분의 시간이 소요될 수 있습니다.
